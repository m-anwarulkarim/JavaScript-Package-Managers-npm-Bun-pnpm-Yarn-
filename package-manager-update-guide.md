আপনি যদি কোনো একটি প্যাকেজ ম্যানেজার (npm, bun, pnpm, yarn) বা এদের মাধ্যমে ইনস্টল করা কোনো লাইব্রেরির ভার্সন আপডেট করতে চান, তবে তার প্রসেস নিচে দেওয়া হলো:

---

### **১. নিজে প্যাকেজ ম্যানেজার আপডেট করতে (Update the Tool itself)**

যদি আপনি এই টুলগুলোর লেটেস্ট ভার্সন পেতে চান:

- **NPM আপডেট:**

```bash
npm install -g npm@latest

```

- **Bun আপডেট:**

```bash
bun upgrade

```

- **pnpm আপডেট:**

```bash
pnpm add -g pnpm

```

- **Yarn আপডেট (Classic):**

```bash
npm install -g yarn

```

---

### **২. প্রজেক্টের লাইব্রেরি আপডেট করতে (Update Project Dependencies)**

আপনার প্রজেক্টের ভেতরে থাকা নির্দিষ্ট কোনো লাইব্রেরি (যেমন: React, Express) আপডেট করার কমান্ড নিচে দেওয়া হলো:

#### **NPM ব্যবহার করলে:**

- নির্দিষ্ট প্যাকেজ: `npm update <package-name>`
- সবগুলো প্যাকেজ: `npm update`

#### **Bun ব্যবহার করলে:**

- নির্দিষ্ট প্যাকেজ: `bun add <package-name>@latest`
- সবগুলো আপডেট করার জন্য বর্তমানে Bun-এ আলাদা `update` কমান্ড নেই, তাই `bun install` দিলেই ইন্টারনালি চেক করে।

#### **pnpm ব্যবহার করলে:**

- ইন্টারেক্টিভ আপডেট (সবচেয়ে সুন্দর পদ্ধতি):

```bash
pnpm update --interactive

```

_(এটি আপনাকে একটি লিস্ট দেখাবে যে কোনটির ভার্সন কত আপডেট হয়েছে, আপনি সিলেক্ট করতে পারবেন)_

#### **Yarn ব্যবহার করলে:**

- নির্দিষ্ট প্যাকেজ: `yarn upgrade <package-name>`
- ইন্টারেক্টিভ আপডেট: `yarn upgrade-interactive --latest`

---

### **৩. একটি প্রো-টিপ (সবগুলোর জন্য কাজ করবে)**

আপনি যদি কোনো টুল বা কমান্ড মনে রাখতে না চান, তবে **`ncu` (npm-check-updates)** নামের একটি অসাধারণ প্যাকেজ ব্যবহার করতে পারেন। এটি সব প্যাকেজ ম্যানেজারের জন্যই কাজ করে।

1. প্রথমে এটি রান করুন:

```bash
npx npm-check-updates -u

```

2. তারপর আপনার পছন্দের কমান্ড দিন:
   `npm install` অথবা `pnpm install` অথবা `bun install` অথবা `yarn`

---

**সতর্কতা:** বড় কোনো ভার্সন আপডেট (যেমন: v1.0.0 থেকে v2.0.0) করার সময় সাবধানে করবেন, কারণ এতে আপনার কোড কাজ করা বন্ধ করে দিতে পারে (Breaking changes)।
